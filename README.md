# api-creditsystem
API for managing creditcard details.

# How to run...

### 1. Clone this repository.

```
git clone git@github.com:AmitSharmaXIII/api-creditsystem.git
```

### 3. Install packages

```
npm install 
```
or
```
yarn install
```

### 4. Run the service

```
npm run ds
```

### 4. Run the service

```
npm run test
```
